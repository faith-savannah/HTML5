Bullet Point Summary of the Book "A Book Apart"
Author: Jeremy Keith

Chapter One: A brief history of Markup

  • HyperText Markup Language(HTML) was the brainchild of Sir Tim Berners-Lee

  • The first official specification was HTML 2.0, published by the IETF.Thus there was never HTML 1.0

  • The role of the IETF was superceded by the W3C, the World Wide Web Consortium, where subsequent iterations of the HTML standard have been published at http://www.w3.org.

  • After subsequent revisions from HTML 2.0 to HTML 4.01, the next revision bore XHTML 1.0

  • eXtreem HTML 1.0/XHTML 1.0 was different from HTML in the syntax only whereby XHTML required a stricter adherance to the rules of XML markup language.

  • WHATWG stands for WebHypertext Application Technology Working Group.It propagated the idea of extending HTML to allow the creation of web applications, opposed to the path the W3C took of XHTML 2.0 which was backward incorpartible.

  • HTML5 is a specification that merged two specifications: Web Forms 2.0 and Web Apps 1.0. Both specifications were from the WHATWG and were intended to extend HTML
